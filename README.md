### _Изучение Объектно ориентированного программирования Java._
    - Использование Java Collections Frameworks;
    - Написание тестов JUnit 4;

## Создание консольной программы для работы с заявками:
                  - Добавление заявки.
                  - Поиск заявки.
                  - Замена заявки.
                  - Удаление заявки.

## Сборка и запуск
+ Сборка Maven
+ Запуск программы StartUI
